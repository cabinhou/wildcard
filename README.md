#### 3、申请WildCard虚拟卡

> 在“我的卡片”中，申请信用卡，服务年限可以选择“2年”或者“3年”，然后使用“支付宝”进行支付。



##### 3.1  服务年限选择和支付宝支付

```sh
#建议选择2年的服务年限，默认是11.99美元，使用支付宝进行支付，由于手续费原因，金额会多一点。使用邀请码KK99可以优惠1美元。
```

```sh
#认证方式为“支付宝实名认证”和“缴押金免实名认证”，考虑到尽量减少信息泄露，建议选择第二种，支付10美元押金。如果每月支付金额较多，那就选择“支付宝”认证，每月可消费3000美元。
```
